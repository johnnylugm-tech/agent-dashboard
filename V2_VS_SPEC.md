## 🎯 下一步建議

### 高優先級（MVP 剩餘）

1. **日誌搜尋** - 接入 Elasticsearch 或 SQLite FTS
2. **執行路徑視覺化** - React 組件
3. **靜音時段** - 簡單配置

4. **RBAC** - 基礎三級權限
5. **Prompt 版本管理** - 簡單版本記錄
6. **Webhook** - 告警觸發

### 低優先級（V1.5+）

7. Docker 部署
8. K8s Helm
9. 多 Provider 整合
